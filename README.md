# INT_SET_T
Библиотека для работы с множествами целых чисел на языке С.
## Индивидуальное задание 
Разработать библиотеку для работы с множествами целых чисел. 
Множество — это фундаментальная структура данных, которая представляет собой набор уникальных элементов, хранящихся в неопределенном порядке.
Реализованы: 
- библиотека int_set_t
- функция добавления жлемента в множество 
- функция удаления элемента из множества 
- функция проверки пренодлежности элемента в множестве
- функция вычисления мощности множества
- функция вычисления пересечения множеств
- функция вычисления симметрической разности
- функция проверки строгого включения одного множества в другое
- функция проверки нестрогого включения одного множества в другое
